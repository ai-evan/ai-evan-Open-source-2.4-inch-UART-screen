Open source of a 2.4-inch serial port screen with free-to-use UI design. This project open-sources the hardware design and schematic design; the screen model is commercially available for purchase on the official website. The firmware can be modified according to specific requirements. Due to a Non-Disclosure Agreement (NDA), the full source code of the firmware cannot be released. For the source code of the UI project, please refer to the project files.
